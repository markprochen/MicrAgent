# 系统监控助手手册 (system_monitor)

用于监控计算机系统的资源使用情况，包括 CPU、内存、磁盘和进程信息。

## 函数列表
1. **get_cpu_info**: `{}`
   - 获取 CPU 核心数、使用率和频率信息
   - 返回物理核心数、逻辑核心数、CPU 使用率和当前频率

2. **get_memory_info**: `{}`
   - 获取内存使用情况
   - 返回总内存、已用内存、可用内存和使用率

3. **get_running_processes**: `{"limit": "返回的进程数量，默认为10"}`
   - 获取当前运行的进程列表
   - 按 CPU 使用率排序，返回指定数量的进程信息
   - 包含 PID、进程名、CPU 和内存占用

4. **get_system_info**: `{}`
   - 获取系统基本信息
   - 返回操作系统版本、主机名、架构和处理器信息

5. **get_disk_info**: `{}`
   - 获取磁盘使用情况
   - 返回所有分区的总容量、已用空间、可用空间和使用率

## 操作规范 (SOP)
- **CPU 监控**: 使用 get_cpu_info 查看 CPU 核心数和使用率，判断系统负载
- **内存监控**: 使用 get_memory_info 查看内存使用情况，判断是否需要释放内存
- **进程监控**: 使用 get_running_processes 查看占用资源最多的进程
- **系统信息**: 使用 get_system_info 了解系统基本配置
- **磁盘监控**: 使用 get_disk_info 查看磁盘空间使用情况

## 数据说明
- CPU 使用率: 采样间隔为 1 秒
- 内存单位: GB (千兆字节)
- 磁盘单位: GB (千兆字节)
- 进程列表: 默认返回 CPU 使用率最高的 10 个进程

## 使用场景
- 系统性能诊断
- 资源占用分析
- 进程管理
- 系统健康检查
