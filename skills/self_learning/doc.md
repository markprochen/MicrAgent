# 技能自进化规范 (self_learning)

## 创建步骤
1. **部署代码**：使用 `write_python_skill` 工具。
   - 只需要提供 `skill_name` 和 `code_content`。
   - code_content 为代码主体内容，不需要生成./skills/weather/__init__.py等类似文件，write_python_skill会自行生成。
   - 系统会自动处理目录创建和 logic.py 写入。
2. **部署文档**：使用 `write_local_file` 写入 `./skills/[skill_name]/doc.md`。
   - 参照标准模板编写说明。
3. **验证**：调用 `get_skill_detail` 确认技能已激活。
4. **避免问题**： 由于目前没办法处理双引号。 所以写入logic.py 和doc.md 两个文件时，不允许带任何双引号，单引号等字符信息

## 核心定义
- **技能学习**：是指通过编写 Python （对应技能名称.py）并配套使用说明（doc.md）来扩展 Agent 能力的过程。
- **物理结构**：每个新技能必须在 `./skills/` 目录下拥有独立文件夹，名称为logic.py。

## 技能创建三步走（强制流程）
1. **目录初始化**：在 `./skills/` 下创建新文件夹。
2. **逻辑实现**：在该文件夹下创建 `logic.py`。
   - 必须使用 Python 编写。
3. **文档定义**：在该文件夹下创建 `doc.md`。
4. **调试**：由于异步加载的问题，目前无法测试你的代码是否有问题。所以可以先把代码存放完整即可
