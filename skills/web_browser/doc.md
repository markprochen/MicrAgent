# 网页深度交互手册 (web_browser)

当 `web_search` 的摘要信息不足以解决问题时，使用此工具深入阅读网页。

## 函数列表
1. **web_browser**: `{"action": "goto|click|type", "url": "网址", "selector": "CSS选择器", "text": "内容"}`
   - `goto`: 访问并提取网页前 2000 字的纯文本。
   - `click`: 点击页面上的按钮或链接。
   - `type`: 在输入框填写内容。

## 操作规范 (SOP)
- **信息提取**：优先执行 `goto` 操作以获取全文进行分析。
- **选择器准则**：在使用 `click` 或 `type` 之前，必须先通过 `goto` 获取到的页面内容来推断正确的 CSS 选择器或元素特征。
- **分段阅读**：如果网页内容极长，Agent 应意识到目前只获取了前 2000 字。
